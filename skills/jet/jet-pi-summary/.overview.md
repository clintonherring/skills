# jet-pi-summary

## Purpose
Investigate production incidents using the Observability Data Lake (ODL) via AWS Athena. This skill responds to four explicit commands: - "pi timeline <channel>" — build a chronological incident timeline - "pi catchup <channel>" — summarise recent discussion and action items - "pi post-mortem <channel>" — generate a pre-filled post-mortem document - "pi prodmeet" — generate the daily production meeting host briefing The channel is a PI channel name like pi-12345. Do NOT run this skill for general questions about incidents — only when the user types "pi timeline", "pi catchup", "pi post-mortem", or "pi prodmeet". metadata: owner: ai-platform
