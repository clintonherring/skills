# jet-event-discovery

## Purpose
"Discovers and maps event-driven architecture across JET (Just Eat Takeaway) internal services, requiring access to JET GitHub Enterprise, BigQuery, Backstage, and Datadog. Produces summaries, tables, and Mermaid diagrams of producers, consumers, and message flows. Use when asked about Kafka topics, SNS/SQS queues, RabbitMQ exchanges, message producers/consumers, event flows, topic ownership, pub/sub, async messaging, dead letter queues, or inter-service communication patterns."
