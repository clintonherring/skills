# jet-github-actions

## Purpose
| JET GitHub Actions skill for writing, debugging, and maintaining CI/CD workflows on GitHub Enterprise Server (GHES). Use this skill when: - Writing or editing GitHub Actions workflow files (.github/workflows/*.yml) - Debugging workflow failures, runner issues, or action version errors - Asking about available actions or reusable workflows - Working with Artifactory, Vault secrets, Docker builds, Helm/Helmfile deployments - Migrating from GitLab, Jenkins, TeamCity, Concourse, or Zuul to GitHub Actions - Asking about runners, runner groups, runner images, or SDK versions - Asking about GHES-specific constraints, imported actions, or version pinning - Setting up GitHub Actions for a new repository or organisation - Working with Kubernetes deployments, AWS credentials, or SonarQube integration
