# jet-aws-athena

## Purpose
AWS Athena query execution and data exploration via the AWS CLI. Use this skill when running SQL queries against Athena, exploring Athena databases, tables, and schemas, checking query execution status, fetching query results, listing workgroups, or when the user asks about querying data in S3 via Athena or references AWS Athena in any context.
