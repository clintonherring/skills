# jet-odl-athena

## Purpose
Query JET's Operational Data Lake (ODL) via AWS Athena to investigate production incidents, track deployments, look up team ownership, review security posture, analyze GitHub activity, and examine reliability and cost metrics. Use this skill when the user asks about recent deployments, production changes, who deployed what, failed deployments, deployment history for an application, team ownership of services, Wiz security issues or vulnerabilities, GitHub PR activity, infrastructure costs, reliability metrics, or any question that can be answered from the ODL. Also triggers on mentions of ODL, operational data lake, DORA metrics, deployment history, or platform metadata lookups.
