# jet-jira-repo-finder

## Purpose
| **What it does:** Given a Jira key, fetches all portfolio children using `portfolioChildIssuesOf` (with pagination), searches GitHub Enterprise for PRs whose title starts with each child ticket key, and returns the unique set of repositories touched by that Jira entity. Uses `acli` for Jira and `gh` for GitHub Enterprise. **When to use it:** - Someone asks which repos are involved in an epic, initiative, or theme - A user wants to map a Jira entity to GitHub repositories - A user asks "what repos does this epic touch?" or "which services are part of this initiative?" - A user asks to "find all repos linked to this Jira key" - A user needs a GitHub footprint for any Jira hierarchy node (epic, initiative, theme, or any ticket with portfolio children)
