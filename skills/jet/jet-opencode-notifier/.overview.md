# jet-opencode-notifier

## Purpose
Set up and configure opencode-notifier for native desktop notifications in OpenCode. Use this skill when the user wants notification setup, OpenCode alerts, opencode-notifier config, Ghostty notifications, or to get notified on task completion, permission requests, errors, or questions.

## Key Sections
- Start by asking
- Install the plugin (beta for Ghostty support)
- Create the notifier config
- Notification system choices
- Validate install
- Remove after setup
- Configuration reference (quick)
- Troubleshooting
