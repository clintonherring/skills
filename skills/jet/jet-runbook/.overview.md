# jet-runbook

## Purpose
Generate comprehensive runbooks for JET (Just Eat Takeaway.com) applications and services. Use when creating documentation that describes service purpose, dependencies, operational procedures, monitoring, alerting, and troubleshooting guides. Triggers on requests to create, write, or generate runbooks for JET services provided the service name, tier, and desired repository path.
