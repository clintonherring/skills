# opensar-nocodb

## Purpose
Browse and manage database records via NocoDB MCP servers. Covers two bases -- OpenSAR (default, cloud_sample + opensar_enhanced) and Tax (opt-in, only when user mentions "tax"). Use when querying, viewing, inserting, updating, or deleting database records, checking device status, managing search parties, verifying wayline data, investigating database issues, or working with tax records.
