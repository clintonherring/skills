# opensar-brand

## Purpose
Applies OpenSAR brand identity (colors, typography, spacing, components) to web frontends and artifacts. Based on Modern Minimalist with aviation-blue SAR accents. Use when styling OpenSAR web components, creating MUI themes, building dashboards, designing UI for the OpenSAR platform, or when brand consistency is needed across OpenSAR frontends.
