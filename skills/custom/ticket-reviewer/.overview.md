# ticket-reviewer

## Purpose
Review Jira tickets that are in Review status assigned to the current user. Use this skill whenever the user asks to review tickets, check their review queue, verify ticket work, or mentions Jira reviews. Also use when the user says "review my tickets", "what tickets need review", "check my review queue", or anything related to verifying completed work on Jira tickets before closing them.

## Key Sections
- Prerequisites
- Step 1: Fetch tickets in review
- Step 2: Review each ticket
- Step 3: Take action
- Self-Improving Context
- Important notes
