# skills

## Purpose
Central repository of portable Cursor/OpenCode agent skills, organized by origin and domain. Each skill is a self-contained SKILL.md with YAML frontmatter (name, description, triggers) and markdown instructions.

## Categories

- **anthropic/** -- Community and Anthropic-authored skills: design system (adapt, animate, audit, bolder, clarify, colorize, critique, delight, distill, extract, harden, normalize, onboard, optimize, polish, quieter, teach-impeccable), document tools (docx, pdf, pptx, xlsx, doc-coauthoring), creative (algorithmic-art, canvas-design, slack-gif-creator, theme-factory), platform (bootstrap-studio-bsdesign, frontend-design, web-artifacts-builder, webapp-testing), developer (claude-api, mcp-builder, skill-creator, skill-reviewer, context-tiering, cost-optimized-routing)
- **cursor/** -- Cursor IDE-specific skills: agent workflow (babysit, canvas, create-hook, create-rule, create-skill, create-subagent, migrate-to-skills, shell, split-to-prs), IDE config (statusline, update-cli-config, update-cursor-settings)
- **jet/** -- JET (Just Eat Takeaway) workplace skills: infrastructure (jet-aws, jet-k8s-debugger, sonic-*, cloudops-*), observability (jet-datadog, jet-pagerduty), data (jet-bq, jet-aws-athena, jet-odl-athena, jet-datahub), workflow (jet-epic-summary, jet-report, jet-runbook, jet-slides, internal-comms)
- **custom/** -- Personal and project-specific skills: OpenSAR (opensar-brand, opensar-nocodb), Android (android-ui-m3), incident response (jet-pi-troubleshooter, ticket-reviewer)
- **toolkit/** -- CLI tool reference docs (acli, aws, gh, pup, wiz)

## Management
- `registry.json` -- Flat index of all skills with abstracts
- `skill-manager.ps1` -- PowerShell tool for listing, syncing, and managing skills
