# cursor

## Purpose
Skills specific to the Cursor IDE agent environment. These extend agent capabilities within Cursor for workflow automation, PR management, and IDE configuration.

## Skills

### Agent Workflow
- babysit -- Keep PRs merge-ready by triaging comments and fixing CI
- canvas -- Create live React apps beside the chat (Cursor Canvas)
- create-hook -- Create Cursor hooks for agent event automation
- create-rule -- Create Cursor rules for persistent AI guidance
- create-skill -- Guide for authoring new Agent Skills
- create-subagent -- Create specialized subagents
- migrate-to-skills -- Migrate existing rules/prompts to skills format
- shell -- Command execution specialist patterns
- split-to-prs -- Split work into small reviewable PRs

### IDE Configuration
- statusline -- Configure custom CLI status line
- update-cli-config -- Modify CLI configuration
- update-cursor-settings -- Modify Cursor/VSCode settings.json
