# cost-optimized-routing

## Purpose
Optimizes token cost by routing tasks to the cheapest effective agent. Classifies work into phases (explore, plan, build, troubleshoot) and delegates to fast subagents for simple work while keeping the main agent as orchestrator for context and escalation. Use at the start of every task, when planning work, when building features, or when the user asks about cost optimization.
